The code is split into a python backend (/server) and a svelte-kit frontend (/client).

# Install it

Server :
https://www.python.org/downloads/

Client:
npm install
nvm use 16.18.0

# Run it

Open 2 terminals.

In the first run :

```
cd client
npm run dev
```

In the second terminal :

```
python3 server/server.py
```
